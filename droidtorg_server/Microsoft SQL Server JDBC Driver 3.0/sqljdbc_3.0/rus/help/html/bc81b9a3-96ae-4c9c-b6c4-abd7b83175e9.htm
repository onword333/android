Метод getBigDecimal (SQLServerCallableStatement)

Возвращает значение указанного параметра в виде значения java.math.BigDecimal.

Список перегрузок

Имя Описание

getBigDecimal (int)

Получает значение указанного параметра в виде объекта java.math.BigDecimal с полной точностью по заданному индексу параметра.

getBigDecimal (int, int)

(Устарело) Получает значение указанного параметра в виде объекта java.math.BigDecimal по заданному индексу параметра и масштабу.

getBigDecimal (java.lang.String)

Получает значение указанного параметра в виде объекта java.math.BigDecimal с полной точностью по заданному имени параметра.

getBigDecimal (java.lang.String, int)

(Устарело) Получает значение указанного параметра в виде объекта java.math.BigDecimal по заданному имени параметра и масштабу.
