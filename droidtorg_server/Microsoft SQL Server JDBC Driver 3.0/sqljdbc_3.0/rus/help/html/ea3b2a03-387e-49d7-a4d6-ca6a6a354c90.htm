Метод truncate (SQLServerClob)

Выполняет усечение объекта CLOB до заданной длины.

Синтаксис

public void truncate(long len)

Параметры

len

Длина в символах, до которой усекается объект CLOB.

Исключения

java.sql.SQLException

Примечания

Этот метод truncate указывается с помощью метода truncate в интерфейсе java.sql.Clob.
