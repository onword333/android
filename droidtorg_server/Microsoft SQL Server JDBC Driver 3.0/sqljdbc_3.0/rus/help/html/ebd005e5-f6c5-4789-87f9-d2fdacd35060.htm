Метод position (java.sql.Blob, long)

Возвращает положение указанного шаблона в большом двоичном объекте на основе указанного шаблона и начального индекса.

Синтаксис

public long position(java.sql.Blob pattern,
                     long start)

Параметры

pattern

Шаблон для поиска.

start

Индекс, с которого начинается поиск.

Возвращаемое значение

Значение типа long, указывающее позицию, в которой обнаружен шаблон, или значение -1, если шаблон не обнаружен.

Исключения

SQLServerException

Примечания

Этот метод position указывается с помощью метода position в интерфейсе java.sql.Blob.
