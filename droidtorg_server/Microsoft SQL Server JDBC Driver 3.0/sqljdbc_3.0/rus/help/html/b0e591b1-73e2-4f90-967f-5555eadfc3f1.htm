Метод isWrapperFor (SQLServerPreparedStatement)

Указывает, является ли этот объект инструкции оболочкой указанного интерфейса.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public boolean isWrapperFor(Class iface)

Параметры

iface

Значение class, определяющее интерфейс.

Возвращаемое значение

Значение true, если этот объект реализует интерфейс или упаковывает объект, реализующий интерфейс. В противном случае значение false.

Исключения

SQLServerException

Примечания

Метод isWrapperFor и метод unwrap определяются с помощью интерфейса java.sql.Wrapper, представленного в спецификации JDBC 4.0.

Если этот метод возвращает значение true, вызов метода unwrap с таким же аргументом будет выполнен успешно.

Дополнительные сведения см. в разделе Оболочки и интерфейсы.
