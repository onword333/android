Элементы SQLServerXAResource

В следующих таблицах перечислены доступные члены класса SQLServerXAResource.

Конструкторы

None.

Поля

Имя Описание

SSTRANSTIGHTLYCPLD

Используется для разрешения тесно связанных транзакций XA, имеющих различные идентификаторы ветвей транзакции (XID), но одинаковые глобальные идентификаторы транзакции (GTRID).

Наследуемые поля

Класс, из которого наследуется Методы

javax.transaction.xa.XAResource

TMENDRSCAN, TMFAIL, TMJOIN, TMNOFLAGS, TMONEPHASE, TMRESUME, TMSTARTRSCAN, TMSUCCESS, TMSUSPEND, XA_OK, XA_RDONLY

Методы

Имя Описание

commit

Фиксирует глобальную транзакцию, которая указана в заданном объекте Xid.

end

Завершает работу, выполняемую от имени ветви транзакции.

forget

Удаляет в диспетчере ресурсов данные об эвристически выполненной ветви транзакции.

getTransactionTimeout

Получает значение времени ожидания текущей транзакции, заданное для этого объекта SQLServerXAResource.

isSameRM

Определяет, совпадает ли экземпляр диспетчера ресурсов, представленный целевым объектом, с экземпляром диспетчера ресурсов, представленным заданным объектом XAResource.

prepare

Запрашивает в диспетчере ресурсов подготовку к фиксации транзакции, указанной в заданном объекте Xid.

recover

Получает список ветвей подготовленной транзакции от диспетчера ресурсов.

rollback

Запрашивает в диспетчере ресурсов откат работы, выполненной от имени ветви транзакции.

setTransactionTimeout

Задает значение времени ожидания текущей транзакции для этого объекта SQLServerXAResource.

start

Начинает работу от имени ветви транзакции, указанной в объекте Xid.
