Метод getSendTimeAsDatetime (SQLServerDataSource)

Этот метод добавлен в версии 3.0 драйвера SQL Server JDBC.

Возвращает значение свойства соединения sendTimeAsDatetime.

Синтаксис

public boolean getSendTimeAsDatetime();

Возвращаемое значение

Значение true, если значения java.sql.Time будут отправляться на сервер с типом SQL Server datetime. Значение false, если значения java.sql.Time будут отправляться на сервер с типом SQL Server time.

Примечания

Дополнительные сведения о свойстве соединения sendTimeAsDatetime см. в разделе Задание свойств соединения.

Метод SQLServerDataSource.setSendTimeAsDatetime позволяет программным образом устанавливать свойство соединения sendTimeAsDatetime.

Дополнительные сведения см. в разделе Настройка способа отправки значений java.sql.Time на сервер.
