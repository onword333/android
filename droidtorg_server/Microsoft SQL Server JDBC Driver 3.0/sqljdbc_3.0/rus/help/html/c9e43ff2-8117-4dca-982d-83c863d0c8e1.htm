Метод setResponseBuffering (SQLServerDataSource)

Задает режим буферизации ответа для соединений, созданных с использованием данного объекта SQLServerDataSource.

Синтаксис

public void setResponseBuffering(java.lang.String value)

Параметры

value

Строка @@@ String, содержащая режим буферизации и потоковой передачи. Допустимый режим может быть одной из следующих строк без учета регистра: full или adaptive.

Примечания

Значение full указывает на чтение всех результатов с сервера во время выполнения.

Значение adaptive указывает на буферизацию необходимого минимума данных. Значение adaptive указывает на режим буферизации по умолчанию для драйверов JDBC Microsoft SQL Server версии 2.0 и более поздних версий.

Дополнительные сведения об использовании режима буферизации ответов см. в разделе Использование адаптивной буферизации.
