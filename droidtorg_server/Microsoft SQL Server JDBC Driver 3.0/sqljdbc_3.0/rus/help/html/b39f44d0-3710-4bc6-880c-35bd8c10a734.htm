Работа с типами данных (JDBC)

Основная функция драйвера JDBC Microsoft SQL Server — доступ к содержимому баз данных SQL Server для разработчиков ПО на Java. С этой целью драйвер JDBC включается в процесс преобразования между типами данных SQL Server и типами и объектами языка Java.

Примечание. С подробным обсуждением типов данных SQL Server и драйвера JDBC, в том числе различий между ними и способов их преобразования в типы данных на языке Java, можно ознакомиться по адресу Основные сведения о типах данных драйвера JDBC.

В целях совместимости с типами данных SQL Server драйвер JDBC поддерживает методы get<Type> и set<Type> для классов SQLServerPreparedStatement и SQLServerCallableStatement, а также методы get<Type> и update<Type> для класса SQLServerResultSet. Применяемый метод зависит от типа обрабатываемых данных, а также от использования либо результирующих наборов, либо запросов.

В подразделах данного раздела описан порядок использования типов данных драйвера JDBC для доступа к данным SQL Server в приложениях Java.
