Определяемые пользователем типы

Определяемые пользователем типы появились в SQL Server 2005, чтобы дать разработчику возможность расширить серверную систему скалярных типов путем хранения объектов среды CLR в базе данных SQL Server. Определяемые пользователем типы могут содержать несколько элементов, и их поведение может отличаться от традиционных псевдонимов типов данных, которые состоят из одного системного типа данных SQL Server. Ранее размер определяемых пользователем типов был ограничен 8 килобайтами. В SQL Server 2008 добавлена поддержка определяемых пользователем типов, размер которых превышает 64 килобайта. В драйвере JDBC 3.0 для SQL Server теперь также поддерживаются определяемые пользователем типы с размером более 64 килобайт при задании формата UserDefined.

Поведение определяемых пользователем типов с размером, не превышающим 8000 байт, не изменилось, но теперь поддерживаются более крупные, определяемые пользователем типы, для которых сообщается размер unlimited.
