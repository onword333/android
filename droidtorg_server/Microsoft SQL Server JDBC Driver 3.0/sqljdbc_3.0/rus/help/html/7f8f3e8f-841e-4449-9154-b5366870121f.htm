Использование инструкций с драйвером JDBC

Драйвер JDBC Microsoft SQL Server можно использовать для работы с данными в базе данных SQL Server несколькими способами. Драйвер JDBC можно использовать для выполнений инструкций SQL для базы данных или вызова хранимых процедур в базе данных, используя как входные, так и выходные параметры. Драйвер JDBC также поддерживает использование escape-последовательностей SQL, счетчиков обновления, автоматически формируемых ключей, а также выполнение обновлений в рамках пакетной операции.

Драйвер JDBC предоставляет три класса для получения данных из базы данных SQL Server.

  1. SQLServerStatement ― используется для выполнения инструкций SQL без параметров.
  2. SQLServerPreparedStatement ― (унаследовано от SQLServerStatement) используется для выполнения компилированных инструкций SQL, которые могут содержать параметры IN.
  3. SQLServerCallableStatement ― (унаследовано от SQLServerPreparedStatement) используется для выполнения хранимых процедур, которые могут содержать параметры IN, OUT либо и те, и другие.

В подразделах данного раздела описывается использование каждого из трех классов инструкций для работы с метаданными в базе данных SQL Server.
