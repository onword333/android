Элементы SQLServerClob

Конструкторы

Имя Описание

SQLServerClob

Инициализирует новый экземпляр класса SQLServerClob.

Поля

None.

Наследуемые поля

None.

Методы

Имя Описание

free

Этот метод освобождает объект CLOB и освобождает занятые им ресурсы.

getAsciiStream

Материализует Clob в виде потока ASCII.

getCharacterStream

Возвращает данные Clob в виде объекта java.io.Reader или потока символов.

getSubString

Возвращает копию указанной подстроки в Clob по указанной начальной позиции и числу символов для копирования.

length

Возвращает число символов в Clob.

position

Возвращает позицию символа указанного объекта Clob или подстроки в Clob по указанной начальной позиции.

setAsciiStream

Возвращает поток для записи символов ASCII в Clob, начиная с указанной позиции.

setCharacterStream

Возвращает поток для записи символов Юникода в Clob, начиная с указанной позиции.

setString

Записывает заданную строку в Clob, начиная с указанной позиции.

truncate

Усекает Clob до указанной длины.
