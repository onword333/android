Метод updateObject (java.lang.String, java.lang.Object, int)

Обновляет указанный столбец значением типа Object по заданному имени столбца и масштабу.

Синтаксис

public void updateObject(java.lang.String columnName,
                         java.lang.Object x,
                         int scale)

Параметры

columnName

Значение String , содержащее имя столбца.

obj

Значение Object .

scale

Для типов java.sql.Types.DECIMAL и java.sql.Types.NUMERIC это количество разрядов после десятичного разделителя. Для всех остальных типов это значение не учитывается.

Исключения

SQLServerException

Примечания

Этот метод updateObject указывается с помощью метода updateObject в интерфейсе java.sql.ResultSet.
