Метод setTime (java.lang.String, java.sql.Time, java.util.Calendar)

Устанавливает указанный параметр в заданные значения времени и календаря.

Синтаксис

public void setTime(java.lang.String sCol,
                    java.sql.Time x,
                    java.util.Calendar c)

Параметры

sCol

Значение типа String, содержащее имя параметра.

x

Объект Time.

c

Объект Calendar.

Исключения

SQLServerException

Примечания

Этот метод setTime указывается методом setTime в интерфейсе java.sql.CallableStatement.

Начиная с версии 3.0 драйвера SQL Server JDBC, на работу этого метода влияет свойство соединения sendTimeAsDatetime (Задание свойств соединения) и SQLServerDataSource.setSendTimeAsDatetime.

Дополнительные сведения см. в разделе Настройка способа отправки значений java.sql.Time на сервер.
