Элементы SQLServerDataSource

В следующих таблицах перечислены члены, предоставляемые классомSQLServerDataSource.

Конструкторы

Имя Описание

SQLServerDataSource ()

Инициализирует новый экземпляр класса SQLServerDataSource.

Поля

None.

Наследуемые поля

None.

Методы

Имя Описание

getApplicationName

Возвращает имя приложения.

getConnection

Пытается установить соединение с источником данных, представляемым данным объектом SQLServerDataSource.

getDatabaseName

Возвращает имя базы данных.

getEncrypt

Возвращает значение boolean, определяющее, включено ли свойство encrypt.

getDescription

Возвращает описание источника данных.

getFailoverPartner

Возвращает имя сервера отработки отказа, используемого в конфигурации зеркального отображения базы данных.

getHostNameInCertificate

Возвращает имя узла, используемого при проверке SSL-сертификата SQL Server.

getInstanceName

Возвращает имя экземпляра SQL Server.

getLastUpdateCount

Возвращает значение boolean, определяющее, включено ли свойство lastUpdateCount.

getLockTimeout

Возвращает значение int, указывающее время в миллисекундах, которое ждет база данных, прежде чем отправить отчет о времени ожидания блокировки.

getLoginTimeout

Возвращает время в секундах, в течение которого данный объект SQLServerDataSource ожидает при попытке установить соединение.

getLogWriter

Возвращает символьный выходной поток, который используется для всех сообщений ведения журнала и трассировки.

getPacketSize

Возвращает текущий размер сетевого пакета в байтах, используемый для обмена данными с SQL Server.

getPortNumber

Возвращает текущий номер порта, используемый для обмена данными с SQL Server.

getReference

Возвращает ссылку на этот объект SQLServerDataSource.

getResponseBuffering

Возвращает режим буферизации ответов для этого объекта SQLServerDataSource.

getSelectMethod

Возвращает тип курсора по умолчанию, используемый для всех результирующих наборов, созданных при помощи данного объекта SQLServerDataSource.

getSendStringParametersAsUnicode

Возвращает значение boolean, определяющее, разрешена ли отправка параметров на сервер в Юникоде.

getSendTimeAsDatetime

Возвращает значение свойства соединения SendTimeAsDatetime.

getServerName

Возвращает имя компьютера, на котором работает SQL Server.

getTrustServerCertificate

Возвращает значение boolean, определяющее, включено ли свойство trustServerCertificate.

getTrustStore

Возвращает путь к файлу сертификата trustStore (включая имя файла).

getURL

Возвращает URL-адрес, используемый для соединения с источником данных.

getUser

Возвращает имя пользователя, используемое для соединения с источником данных.

getUseSQLServerBaseDate

Возвращает значение свойства соединения useSQLServerBaseDate.

getWorkstationID

Возвращает имя клиентского компьютера, используемого для соединения с источником данных.

getXopenStates

Возвращает значение boolean, определяющее, включено ли преобразование состояний SQL в состояния, совместимые с XOPEN.

isWrapperFor

Указывает, является ли этот объект источника данных оболочкой указанного интерфейса.

setApplicationName

Задает имя приложения.

setDatabaseName

Задает имя базы данных для соединения.

setDescription

Задает описание источника данных.

setEncrypt

Задает значение boolean, определяющее, включено ли свойство encrypt.

setFailoverPartner

Задает имя сервера отработки отказа, используемого в конфигурации зеркального отображения базы данных.

setHostNameInCertificate

Задает имя узла, используемого для проверки SSL-сертификата SQL Server.

setInstanceName

Задает имя экземпляра SQL Server.

setIntegratedSecurity

Задает значение boolean, определяющее, включено ли свойство integratedSecurity.

setLastUpdateCount

Задает значение boolean, определяющее, включено ли свойство lastUpdateCount.

setLockTimeout

Задает значение int, определяющее время в миллисекундах, по истечении которого база данных сообщает об истечении времени ожидания блокировки.

setLoginTimeout

Задает время в секундах, которое ждет объект SQLServerDataSource при попытке установить соединение.

setLogWriter

Задает символьный выходной поток, который используется для всех сообщений ведения журнала и трассировки.

setPacketSize

Задает текущий размер сетевого пакета в байтах, используемый для обмена данными с SQL Server.

setPassword

Задает пароль, используемый для соединения с SQL Server.

setPortNumber

Задает номер порта, используемый для обмена данными с SQL Server.

setResponseBuffering

Задает режим буферизации ответа для соединений, созданных с использованием данного объекта SQLServerDataSource.

setSelectMethod

Задает тип курсора по умолчанию, используемый для всех результирующих наборов, созданных при помощи данного объекта SQLServerDataSource.

setSendStringParametersAsUnicode

Задает значение boolean, определяющее, разрешена ли отправка параметров на сервер в Юникоде.

setSendTimeAsDatetime

Задает порядок отправки значений java.sql.Time на сервер.

setServerName

Задает имя компьютера, на котором работает SQL Server.

setTrustServerCertificate

Задает значение boolean, определяющее, включено ли свойство trustServerCertificate.

setTrustStore

Задает путь к файлу сертификата trustStore (включая имя файла).

setTrustStorePassword

Задает пароль, используемый для проверки целостности данных trustStore.

setURL

Задает URL-адрес, используемый для соединения с источником данных.

setUser

Задает имя пользователя, используемое для соединения с источником данных.

setWorkstationID

Задает имя клиентского компьютера, используемого для соединения с источником данных.

setXopenStates

Задает значение boolean, определяющее, включено ли преобразование состояний SQL в состояния, совместимые с XOPEN.

unwrap

Возвращает объект, в котором реализован указанный интерфейс для разрешения доступа к методам, относящимся к драйверу JDBC Microsoft SQL Server.
