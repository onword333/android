Метод updateNClob (int, java.io.Reader, long)

Обновляет заданный столбец с помощью указанного объекта Reader, имеющего указанную длину в символах.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public void updateNClob(int columnIndex,
                        java.io.Reader reader,
                        long length)

Параметры

columnIndex

Значение int, указывающее индекс столбца.

reader

Объект Reader.

length

Количество символов в данных параметра.

Исключения

SQLServerException

Примечания

Этот метод updateNClob указывается с помощью метода updateNClob в интерфейсе java.sql.ResultSet.

Этот метод поддерживается только для столбцов nvarchar(max), ntext и xml. Использование этого метода при работе со столбцами других типов данных приведет к возникновению исключения.
