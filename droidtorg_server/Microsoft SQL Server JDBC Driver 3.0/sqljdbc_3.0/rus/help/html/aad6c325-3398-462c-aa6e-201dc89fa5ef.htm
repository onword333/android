Метод getPooledConnection ()

Устанавливает физическое соединение с базой данных, которое будет использоваться в пулах соединений.

Синтаксис

public javax.sql.PooledConnection getPooledConnection()

Возвращаемое значение

Объект SQLServerPooledConnection.

Исключения

java.sql.SQLException

Примечания

Этот метод getPooledConnection указывается в методе getPooledConnection в интерфейсе javax.sql.ConnectionPoolDataSource.
