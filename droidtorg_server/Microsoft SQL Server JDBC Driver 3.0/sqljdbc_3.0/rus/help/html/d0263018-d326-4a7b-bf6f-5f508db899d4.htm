Метод updateBlob (int, java.io.InputStream)

Обновляет указанный столбец с помощью заданного входного потока.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public void updateBlob(int columnIndex,
                       java.io.InputStream inputStream)

Параметры

columnIndex

Значение int, указывающее индекс столбца.

inputStream

Объект InputStream.

Исключения

SQLServerException

Примечания

Этот метод updateBlob указывается с помощью метода updateBlob в интерфейсе java.sql.ResultSet.
