Метод setCharacterStream (int, java.io.Reader, long)

Устанавливает для параметра значение заданного объекта java.io.Reader, имеющего указанную длину в символах.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public final void setCharacterStream(int parameterIndex,
                                    java.io.Reader reader,
                                    long length)

Параметры

parameterIndex

Значение int, определяющее номер параметра.

reader

Объект java.io.Reader, содержащий данные в Юникоде.

length

Значение long, определяющее число символов в потоке.

Исключения

SQLServerException

Примечания

Этот метод setCharacterStream указывается с помощью метода setCharacterStream в интерфейсе java.sql.PreparedStatement.

Если длина потока отличается от указанной в параметре length, то драйвер JDBC вызовет исключение при обновлении или вставке строки.

Если длина потока неизвестна, то для параметра length может быть задано значение -1, после чего драйвер будет принимать потоки независимо от их длины. Для sqljdbc4.jar рекомендуется использовать метод JDBC 4.0 Метод setCharacterStream (int, java.io.Reader), при котором приложения обновляют столбец с помощью потока, длина которого неизвестна.
