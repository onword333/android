Метод updateAsciiStream (int, java.io.InputStream, int)

Обновляет значение ASCII-потока в указанном столбце, в котором указывается заданное число байтов.

Синтаксис

public void updateAsciiStream(int index,
                              java.io.InputStream x,
                              int length)

Параметры

index

Значение int, указывающее индекс столбца.

x

Объект InputStream.

length

Значение int, указывающее длину потока.

Исключения

SQLServerException

Примечания

Этот метод updateAsciiStream указывается с помощью метода updateAsciiStream в интерфейсе java.sql.ResultSet.

Этот метод передает символы в кодировке ASCII (байты) из объекта InputStream в столбцы символьных значений, поддерживающие преобразование, то есть содержащие символы Юникода в диапазоне ASCII (от 0x00 до 0x7F), а также принадлежащие кодовым страницам 874, 932, 936, 949, 950 и с 1250 по 1258. Этот метод выполняет преобразование на целевую страницу параметров сортировки. Попытка обновления целевого столбца, не поддерживающего преобразование, приведет к возникновению исключения. В столбцах двоичных значений передаются необработанные байты.

Если длина потока отличается от указанной в параметре length, то драйвер JDBC вызовет исключение при обновлении или вставке строки.

Если длина потока неизвестна, то для параметра length может быть задано значение -1, после чего драйвер будет принимать потоки независимо от их длины. Для sqljdbc4.jar рекомендуется использовать метод JDBC 4.0 Метод updateAsciiStream (int, java.io.InputStream), при котором приложения обновляют столбец с помощью потока, длина которого неизвестна.
