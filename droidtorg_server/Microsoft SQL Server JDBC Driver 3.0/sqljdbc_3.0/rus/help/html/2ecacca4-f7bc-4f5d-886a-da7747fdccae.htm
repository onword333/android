Метод cancelRowUpdates (ISQLServerResultSet)

Отменяет обновления текущей строки в этом объекте SQLServerResultSet.

Синтаксис

public void cancelRowUpdates()

Исключения

SQLServerException

Примечания

Этот метод cancelRowUpdates указывается с помощью метода cancelRowUpdates в интерфейсе java.sql.ResultSet.

Этот метод можно вызывать после вызова метода адаптера и перед вызовом метода updateRow для отката обновлений, выполненных в строке. Если обновления не выполнены или уже вызван метод updateRow, то этот метод не выполняет никаких действий.
