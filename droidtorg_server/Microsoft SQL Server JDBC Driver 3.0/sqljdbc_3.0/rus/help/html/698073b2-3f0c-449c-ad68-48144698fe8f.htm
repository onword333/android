Метод setString (long, java.lang.String) (SQLServerNClob)

Записывает указанное значение String в NCLOB, начиная с указанной позиции.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public int setString(long pos,
              java.lang.String str)

Параметры

pos

Позиция, с которой начнется запись в NCLOB; отсчитывается от 1.

str

Значение String, записываемое в NCLOB.

Возвращаемое значение

Количество записанных символов.

Исключения

SQLServerException

Примечания

Этот метод setString указывается с помощью метода setString в интерфейсе java.sql.NClob.
