Метод valueOf (java.sql.Timestamp, java.util.Calendar)

Создает объект DateTimeOffset, представляющий точку во времени с определенным смещением относительно среднего времени по Гринвичу, заданного значением java.sql.Timestamp и значением java.util.Calendar, указывающим смещение.

Синтаксис

public static DateTimeOffset valueOf(java.sql.Timestamp timestamp, java.util.Calendar calendar)

Параметры

timestamp

Значение типа java.sql.Timestamp.

calendar

Значение смещения.

Возвращаемое значение

Возвращает объект DateTimeOffset, представляющий точку во времени, заданную объектом java.sql.Timestamp во временной зоне, определенной объектом java.util.Calendar.

Примечания

Этот метод также задает для объекта java.util.Calendar значение точки во времени, заданное объектом java.sql.Timestamp.
