Метод createStatement (int, int, int)

Создает объект SQLServerStatement, который создает объекты SQLServerResultSet с заданным типом, видом параллелизма и возможностью сохранения.

Синтаксис

public java.sql.Statement createStatement(int nType,
                                          int nConcur,
                                          int nHold)

Параметры

resultSetType

Значение типа int, представляющее тип результирующего набора.

nConcur

Значение типа int, представляющее тип параллелизма результирующего набора.

nHold

Значение типа int, представляющее возможность сохранения.

Возвращаемое значение

Объект Statement.

Исключения

SQLServerException

Примечания

Этот метод createStatement указывается методом createStatement в интерфейсе java.sql.Connection.
