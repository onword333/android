Метод executeUpdate (ISQLServerPreparedStatement)

Выполняет в этом объекте SQLServerPreparedStatement инструкцию SQL. Это должна быть инструкция SQL INSERT, UPDATE, MERGE или DELETE либо инструкция SQL, не возвращающая значения, такая как инструкция DDL.
