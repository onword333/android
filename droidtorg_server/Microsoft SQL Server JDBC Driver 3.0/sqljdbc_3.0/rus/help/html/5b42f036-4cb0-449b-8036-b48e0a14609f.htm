Метод position (SQLServerNClob)

Извлекает положение символа указанного объекта NClob или подстроки в NClob на основании указанной начальной позиции.

Имя Описание

Метод position (java.sql.NClob, long)

Возвращает позицию символа, по которой указанный объект NClob searchstr находится в данном объекте NClob.

Метод position (java.lang.String, long) (SQLServerNClob)

Возвращает позицию символа, в которой указанная подстрока searchstr находится в значении NCLOB, представленном данным объектом NClob.
