Интерфейс ISQLServerCallableStatement

Представляет вызываемые инструкции JDBC. Этот интерфейс добавлен в версии 3.0 драйвера SQL Server JDBC.

Пакет: com.microsoft.sqlserver.jdbc

Расширяет: java.sql.CallableStatement, ISQLServerPreparedStatement

Синтаксис

public interface ISQLServerCallableStatement

Примечания

Этот интерфейс реализуется классом Класс SQLServerCallableStatement.

Этот интерфейс предоставляет следующие методы SQL Server, относящиеся к драйверу JDBC.
