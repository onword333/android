Метод getDateTimeOffset(java.lang.string) (SQLServerResultSet)

Этот метод добавлен в версии 3.0 драйвера Microsoft SQL Server JDBC.

Получает значение заданного столбца в виде объекта Класс DateTimeOffset на языке программирования Java по индексу, заданному параметром.

Синтаксис

public microsoft.sql.DateTimeOffset getDateTimeOffset(String columnName)

Параметры

columnName

Имя столбца.

Возвращаемое значение

Объект Класс DateTimeOffset.

Исключения

SQLServerException

Примечания

Значение Класс DateTimeOffset можно обновить с помощью SQLServerResultSet.updateDateTimeOffset.
