Метод setDateTimeOffset (SQLServerCallableStatement)

Этот метод добавлен в версии 3.0 драйвера Microsoft SQL Server JDBC.

Устанавливает указанный столбец в значение Класс DateTimeOffset.

Синтаксис

public void setDateTimeOffset(String sCol, microsoft.sql.DateTimeOffset t)

Параметры

sCol

Имя столбца.

t

Объект Класс DateTimeOffset.

Исключения

SQLServerException

Примечания

Значение Класс DateTimeOffset можно получить с помощью метода SQLServerCallableStatement.getDateTimeOffset.

setDateTimeOffset принимает порядковый номер столбца.
