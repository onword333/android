Метод removeConnectionEventListener (SQLServerPooledConnection)

Удаляет заданный прослушиватель событий.

Синтаксис

public void removeConnectionEventListener(javax.sql.ConnectionEventListener listener)

Параметры

listener

Объект ConnectionEventListener.

Примечания

Этот метод removeConnectionEventListener указывается в методе removeConnectionEventListener в интерфейсе javax.sql.PooledConnection.
