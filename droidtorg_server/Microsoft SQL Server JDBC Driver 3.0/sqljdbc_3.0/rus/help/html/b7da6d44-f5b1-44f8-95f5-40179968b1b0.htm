Метод setNString (int, java.lang.String)

Задает указанный параметр для заданного объекта String.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public final void setNString(int parameterIndex,
                                                  java.lang.String value)

Параметры

parameterIndex

Значение int, указывающее индекс параметра.

value

Объект String, содержащий значение параметра.

Исключения

SQLServerException

Примечания

Этот метод должен использоваться для типов данных NCHAR, NVARCHAR, NTEXT и XML.

Этот метод setNString указывается с помощью метода setNString в интерфейсе java.sql.PreparedStatement.
