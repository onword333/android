Метод executeQuery (java.lang.String)

Выполняет заданную инструкцию SQL и возвращает единичный объект SQLServerResultSet.

Синтаксис

public final java.sql.ResultSet executeQuery(java.lang.String sql)

Параметры

sql

Значение String, содержащее инструкцию SQL.

Возвращаемое значение

Объект SQLServerResultSet.

Исключения

SQLServerException

Примечания

Этот метод executeQuery указывается методом executeQuery в интерфейсе java.sql.Statement.

Этот метод переопределяет метод executeQuery, содержащийся в классе SQLServerStatement.

Вызов этого метода приводит к исключению, поскольку инструкция SQL для объекта SQLServerPreparedStatement указана при создании объекта.

Исключение SQLServerException вызывается, если заданная инструкция SQL дает любой результат, кроме одиночного объекта SQLServerResultSet.
