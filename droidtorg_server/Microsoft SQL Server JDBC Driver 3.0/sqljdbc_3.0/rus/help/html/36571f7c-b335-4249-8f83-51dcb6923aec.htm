Метод getNClob (java.lang.String) (ISQLServerResultSet)

Извлекает значение заданного столбца в текущей строке объекта SQLServerResultSet в виде объекта NClob на языке программирования Java.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public java.sql.NClob getNClob(java.lang.String columnLabel)

Параметры

columnLable

Значение String, содержащее метку столбца.

Возвращаемое значение

Объект NClob.

Исключения

SQLServerException

Примечания

Этот метод getNClob указывается с помощью метода getNClob в интерфейсе java.sql.ResultSet.

Этот метод поддерживается только для столбцов nvarchar(max), ntext и xml. Использование этого метода при работе со столбцами других типов данных приведет к возникновению исключения.
