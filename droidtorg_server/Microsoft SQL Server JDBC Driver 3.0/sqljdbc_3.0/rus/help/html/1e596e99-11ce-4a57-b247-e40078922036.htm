Метод updateShort (java.lang.String, short)

Обновляет указанный столбец значением типа short по заданному имени столбца.

Синтаксис

public void updateShort(java.lang.String columnName,
                        short x)

Параметры

columnName

Значение String, содержащее имя столбца.

x

Значение типа short.

Исключения

SQLServerException

Примечания

Этот метод updateShort указывается с помощью метода updateShort в интерфейсе java.sql.ResultSet.
