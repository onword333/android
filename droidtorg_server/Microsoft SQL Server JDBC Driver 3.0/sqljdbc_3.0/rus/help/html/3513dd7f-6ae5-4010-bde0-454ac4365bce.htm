Метод getUser (SQLServerDataSource)

Возвращает имя пользователя, используемое для соединения с источником данных.

Синтаксис

public java.lang.String getUser()

Возвращаемое значение

Значение типа String, содержащее имя пользователя.

Примечания

Метод setUser задает имя пользователя, которое будет использоваться при соединении с экземпляром SQL Server. Если значение имени пользователя не задано, метод getUser возвращает значение NULL по умолчанию.
