Методы

Имя Описание

free

Этот метод освобождает большой двоичный объект и ресурсы, занятые им.

getBinaryStream

Возвращает входной поток для чтения данных из большого двоичного объекта.

getBytes

Получает данные большого двоичного объекта в виде массива байтов.

length

Возвращает число байтов в большом двоичном объекте.

position

Возвращает положение указанного шаблона в большом двоичном объекте на основе указанного шаблона и начального индекса.

setBinaryStream

Извлекает поток, который можно использовать для записи значения большого двоичного объекта.

setBytes

Возвращает указанный массив байтов в большой двоичный объект, начиная с указанной позиции, а затем возвращает число записанных байтов.

truncate

Выполняет усечение большого двоичного объекта до заданной длины.
