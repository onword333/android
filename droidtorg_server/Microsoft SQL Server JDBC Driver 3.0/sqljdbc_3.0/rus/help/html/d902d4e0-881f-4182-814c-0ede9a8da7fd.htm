Метод registerOutParameter (int, int, int)

Регистрирует параметр OUT с указанным порядковым номером для заданного типа и масштаба JDBC.

Синтаксис

public void registerOutParameter(int index,
                                 int sqlType,
                                 int scale)

Параметры

index

Значение типа int, указывающее порядковый номер параметра.

sqlType

Код типа JDBC, определенный в java.sql.Types.

scale

Значение типа int, указывающее количество разрядов, размещаемых справа от десятичного разделителя.

Исключения

SQLServerException

Примечания

Этот метод registerOutParameter указывается методом registerOutParameter в интерфейсе java.sql.CallableStatement.

Начиная с версии 3.0 драйвера SQL Server JDBC, если параметр sqlType имеет тип java.sql.Types.TIME, то на работу этого метода влияет свойство соединения sendTimeAsDatetime (Задание свойств соединения) и SQLServerDataSource.setSendTimeAsDatetime.

Дополнительные сведения см. в разделе Настройка способа отправки значений java.sql.Time на сервер.
