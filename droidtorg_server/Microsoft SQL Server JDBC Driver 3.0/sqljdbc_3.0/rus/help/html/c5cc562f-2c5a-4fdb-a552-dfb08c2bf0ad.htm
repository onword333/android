Метод getObject (SQLServerCallableStatement)

Возвращает значение указанного параметра в виде объекта на языке программирования Java.

Список перегрузок

Имя Описание

getObject (int)

Получает значение заданного параметра в виде объекта на языке программирования Java по заданному индексу параметра.

getObject (int, java.util.Map)

Извлекает с помощью указанного объекта Map значение заданного параметра в виде объекта на языке программирования Java при наличии заданного индекса параметров.

getObject (java.lang.String)

Получает значение заданного параметра в виде объекта на языке программирования Java по заданному имени параметра.

getObject (java.lang.String, java.util.Map)

Возвращает значение указанного параметра в виде объекта на языке программирования Java по имени параметра, используя указанный объект Map.
