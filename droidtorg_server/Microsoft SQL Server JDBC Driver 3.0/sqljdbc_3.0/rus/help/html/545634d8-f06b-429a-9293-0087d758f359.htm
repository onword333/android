Метод getParameterClassName (SQLServerParameterMetaData)

Возвращает полное имя класса Java, экземпляры которого должны передаваться методу setObject класса SQLServerPreparedStatement.

Синтаксис

public java.lang.String getParameterClassName(int param)

Параметры

param

Значение типа int, указывающее индекс параметра.

Возвращаемое значение

Значение типа String, содержащее полное имя класса.

Исключения

SQLServerException

Примечания

Этот метод getParameterClassName указывается в методе getParameterClassName в интерфейсе java.sql.ParameterMetaData.
