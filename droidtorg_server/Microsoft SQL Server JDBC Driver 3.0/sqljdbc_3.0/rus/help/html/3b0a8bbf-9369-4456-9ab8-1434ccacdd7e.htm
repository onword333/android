Метод isValid (ISQLServerConnection)

Указывает, верно ли, что данный объект SQLServerConnection не закрыт и до сих пор действителен.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public boolean isValid(int timeout)

Параметры

timeout

Значение int, определяющее время ожидания (в секундах) до выполнения проверки соединения.

Возвращаемое значение

true, если подключение является допустимым; false, если подключение не является допустимым или допустимость соединения не удается определить до завершения периода ожидания.

Исключения

SQLServerException

Примечания

Этот метод isValid указывается методом isValid в интерфейсе java.sql.Connection.
