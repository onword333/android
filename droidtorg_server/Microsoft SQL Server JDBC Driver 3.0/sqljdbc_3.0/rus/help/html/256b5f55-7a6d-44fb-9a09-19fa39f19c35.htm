Метод setClob (java.lang.String, java.sql.Clob)

Задает указанный параметр для заданного объекта Clob.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public final void setClob(java.lang.String parameterName,
            java.sql.Clob x)

Параметры

parameterName

Значение String, содержащее имя параметра.

x

Объект Clob.

Исключения

SQLServerException

Примечания

Этот метод setClob указывается с помощью метода setClob в интерфейсе java.sql.CallableStatement.
