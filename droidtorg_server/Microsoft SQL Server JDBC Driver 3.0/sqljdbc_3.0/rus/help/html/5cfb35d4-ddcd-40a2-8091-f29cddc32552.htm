Обработка метаданных с использованием драйвера JDBC

Драйвер Microsoft SQL Server JDBC можно использовать для работы с метаданными в базе данных SQL Server разнообразными способами. Драйвер JDBC можно использовать для получения метаданных о базе данных, результирующем наборе или параметрах.

Драйвер JDBC предоставляет три класса для получения метаданных из базы данных SQL Server.

  • SQLServerDatabaseMetaData, который используется для возвращения сведений о базе данных, с которой установлено соединение.
  • SQLServerResultSetMetaData, который используется для возвращения сведений о результирующем наборе.
  • SQLServerParameterMetaData, который используется для возвращения сведений о параметрах подготовленных и вызываемых инструкций.

Здесь представлены разделы, где описывается использование каждого из трех классов метаданных для работы с метаданными в базе данных SQL Server.

Примечание. Методы метаданных, описываемые в этом разделе, обычно предъявляют серьезные требования к производительности приложения, поэтому их нужно применять с осторожностью.
