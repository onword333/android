Метод getNString (int)

Извлекает значение указанного параметра NCHAR, NVARCHAR и LONGNVARCHAR в виде объекта String на языке программирования Java.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public final java.lang.String getNString(int parameterIndex)

Параметры

parameterIndex

Значение int, указывающее индекс параметра.

Возвращаемое значение

Объект String .

Исключения

SQLServerException

Примечания

Этот метод getNString указывается с помощью метода getNString в интерфейсе java.sql.CallableStatement.
