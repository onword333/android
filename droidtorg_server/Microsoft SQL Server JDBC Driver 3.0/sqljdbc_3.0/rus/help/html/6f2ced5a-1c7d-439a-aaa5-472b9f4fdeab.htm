Метод updateArray (java.lang.String, java.sql.Array)

Обновляет указанный столбец объектом Array по заданному имени столбца.

Синтаксис

public void updateArray(java.lang.String columnName,
                        java.sql.Array x)

Параметры

columnName

Значение String, содержащее имя столбца.

x

Объект Array.

Исключения

SQLServerException

Примечания

Этот метод updateArray указывается с помощью метода updateArray в интерфейсе java.sql.ResultSet.
