Класс SQLServerConnection

Представляет соединение JDBC с базой данных Microsoft SQL Server.

Пакет: com.microsoft.sqlserver.jdbc

Реализует: ISQLServerConnection, java.io.Serializable

Синтаксис

public class SQLServerConnection

Примечания

Интерфейс SQLServerConnection поддерживает пулы соединений JDBC и может представлять физическое соединение JDBC или логическое соединение JDBC. Интерфейс SQLServerConnection управляет транзакциями для всех инструкций, созданных по этому интерфейсу, и может участвовать в распределенных транзакциях XA, управляемых через адаптер XAResource.

SQLServerConnection управляет пулом дескрипторов подготовленных инструкций. Подготовленные инструкции готовятся один раз и обычно выполняются несколько раз с различными значениями параметров. Кроме того, подготовленные транзакции сохраняются после закрытия логических соединений (входящих в пул).

Примечание. Интерфейс SQLServerConnection не является потокобезопасным. Однако несколько инструкций, созданных из одного соединения, можно обрабатывать одновременно в параллельных потоках.

Данный класс поддерживает развертывание в класс SQLServerConnection и интерфейсы java.sql.connection и ISQLServerConnection. Дополнительные сведения см. в разделе Оболочки и интерфейсы.
