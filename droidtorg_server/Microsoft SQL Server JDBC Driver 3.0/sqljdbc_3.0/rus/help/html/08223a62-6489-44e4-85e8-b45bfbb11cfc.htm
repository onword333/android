Метод setLong (ISQLServerPreparedStatement)

Устанавливает указанный параметр в заданное значение типа long.

Синтаксис

public final void setLong(int n,
                          long x)

Параметры

n

Значение int, указывающий номер параметра.

x

Значение типа long.

Исключения

SQLServerException

Примечания

Этот метод setLong указывается методом setLong в интерфейсе java.sql.PreparedStatement.
