Метод setBoolean (SQLServerCallableStatement)

Присваивает указанному параметру заданное значение boolean.

Синтаксис

public void setBoolean(java.lang.String sCol,
                       boolean b)

Параметры

sCol

Значение типа String , содержащее имя параметра.

b

Значение типа boolean — true или false.

Исключения

SQLServerException

Примечания

Этот метод setBoolean указывается методом setBoolean в интерфейсе java.sql.CallableStatement.
