Метод getMajorVersion (SQLServerDriver)

Возвращает основной номер версии драйвера JDBC Microsoft SQL Server.

Синтаксис

public int getMajorVersion()

Возвращаемое значение

Значение int, который содержит основной номер версии драйвера JDBC.

Примечания

Этот метод getMajorVersion указывается с помощью метода getMajorVersion в интерфейсе java.sql.Driver.
