Поле SSTRANSTIGHTLYCPLD (SQLServerXAResource)

Используется для разрешения тесно связанных транзакций XA, имеющих различные идентификаторы ветвей транзакции (XID), но одинаковые глобальные идентификаторы транзакции (GTRID).

Синтаксис

public static final int SSTRANSTIGHTLYCPLD

Значение поля

Значение 32768 типа int.

Примечания

Каждая транзакция определяется по идентификатору ветви транзакции (XID) и глобальному идентификатору транзакции (GTRID). Чтобы разрешить приложениям использовать тесно связанные транзакции XA, имеющие различные идентификаторы XID, но одинаковые идентификаторы GTRID, необходимо задать значение SSTRANSTIGHTLYCPLD в параметре flags метода XAResource.start. Дополнительные сведения об использовании этого флага см. в разделе Основные сведения о транзакциях XA.
