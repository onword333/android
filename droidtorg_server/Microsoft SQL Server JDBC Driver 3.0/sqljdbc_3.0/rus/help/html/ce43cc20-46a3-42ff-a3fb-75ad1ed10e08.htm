Соединение и извлечение данных

При работе с драйвером JDBC Microsoft SQL Server есть два основных способа установки связи с базой данных SQL Server. Во-первых, можно задать свойства подключения в URL-адресе соединения, затем вызвать метод getConnection класса DriverManager для возвращения объекта SQLServerConnection.

Примечание. Список свойств соединения, поддерживаемых драйвером JDBC, см. в разделе Задание свойств соединения.

По второму методу свойства соединения задаются посредством использования методов задания класса SQLServerDataSource и последующего вызова метода getConnection для возвращения объекта SQLServerConnection.

В подразделах данного раздела описываются различные способы соединения с базой данных SQL Server, а также демонстрируются различные методики извлечения данных.
