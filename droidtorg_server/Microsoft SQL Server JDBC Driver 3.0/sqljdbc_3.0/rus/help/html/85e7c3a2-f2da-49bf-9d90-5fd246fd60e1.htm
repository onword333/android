Метод executeUpdate (java.lang.String) (ISQLServerStatement)

Выполняет заданную инструкцию SQL, которой может быть инструкция INSERT, UPDATE или DELETE; или инструкция SQL, не возвращающая значений, например инструкция SQL DDL. Начиная с версии 3.0 драйвера Microsoft SQL Server JDBC, метод executeUpdate возвращает правильное количество строк, обновленных в операции MERGE.

Синтаксис

public int executeUpdate(java.lang.String sql)

Параметры

sql

Значение типа String, содержащее инструкцию SQL.

Возвращаемое значение

Значение int, которое указывает число обработанных строк, либо значение 0 для инструкций языка определения данных.

Исключения

SQLServerException

Примечания

Этот метод executeUpdate указывается методом executeUpdate в интерфейсе java.sql.Statement.
