Метод isReadOnly (ISQLServerConnection)

Указывает, находится ли данный объект SQLServerConnection в режиме только для чтения.

Примечание. В настоящее время этот метод не поддерживается драйвером JDBC Microsoft SQL Server.

Синтаксис

public boolean isReadOnly()

Возвращаемое значение

Значение true, если соединение выполняется в режиме только для чтения; значение false, если это не так.

Исключения

SQLServerException

Примечания

Этот метод isReadOnly указывается методом isReadOnly в интерфейсе java.sql.Connection.
