Использование инструкции SQL без параметров

Для работы с данными в базе данных SQL Server с использованием инструкции SQL, не содержащей параметров, можно использовать метод executeQuery класса SQLServerStatement для возврата набора SQLServerResultSet, который будет содержать запрошенные данные. Чтобы это сделать, сначала надо создать объект SQLServerStatement, используя метод createStatement класса SQLServerConnection.

В следующем примере открытое соединение с образцом базы данных SQL Server 2005 AdventureWorks передается в функцию, создается и запускается инструкция SQL, и затем результаты считываются из результирующего набора. 

public static void executeStatement(Connection con) {
   try {
      String SQL = "SELECT LastName, FirstName FROM Person.Contact ORDER BY LastName";
      Statement stmt = con.createStatement();
      ResultSet rs = stmt.executeQuery(SQL);

      while (rs.next()) {
         System.out.println(rs.getString("LastName") + ", " + rs.getString("FirstName"));
      }
      rs.close();
      stmt.close();
   }
   catch (Exception e) {
      e.printStackTrace();
   }
}
