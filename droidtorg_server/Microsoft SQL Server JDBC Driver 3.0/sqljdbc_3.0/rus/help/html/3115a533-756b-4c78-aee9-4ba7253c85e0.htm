Элементы ISQLServerConnection

В следующих таблицах перечислены элементы, предоставляемые классом SQLServerConnection.

Конструкторы

None.

Поля

Имя Описание

TRANSACTION_SNAPSHOT

Используется для указания уровня изоляции транзакций моментальных снимков.

Наследуемые поля

Класс, из которого наследуется: Описание

java.sql.Connection

TRANSACTION_NONE, TRANSACTION_READ_COMMITTED, TRANSACTION_READ_UNCOMMITTED, TRANSACTION_REPEATABLE_READ, TRANSACTION_SERIALIZABLE

Методы

Имя Описание

clearWarnings

Удаляет все предупреждения, включенные в отчет для этого объекта SQLServerConnection.

close

Немедленно высвобождает базу данных для этого объекта SQLServerConnection и ресурсы JDBC вместо ожидания их автоматического освобождения.

commit

Фиксирует все изменения, выполненные после предыдущей операции фиксации или отката, и снимает в базе данных все блокировки, удерживаемые в настоящее время данным объектом SQLServerConnection.

createBlob

Создает объект java.sql.Blob, не содержащий данных.

createClob

Создает объект java.sql.Clob, не содержащий данных.

createNClob

Создает объект java.sql.NClob, не содержащий данных.

createStatement

Создает объект SQLServerStatement для отправки инструкций SQL в базу данных.

createSQLXML

Создает объект java.sql.SQLXML, не содержащий данных.

getAutoCommit

Возвращает текущий режим автоматической фиксации для данного объекта SQLServerConnection.

getCatalog

Возвращает текущее имя каталога для данного объекта SQLServerConnection.

getClientInfo

Извлекает дополнительные сведения о свойствах данных клиентов, поддерживаемых драйвером JDBC.

getHoldability

Возвращает текущую возможность сохранения объектов SQLServerResultSet, созданных с помощью данного объекта SQLServerConnection.

getMetaData

Возвращает объект SQLServerDatabaseMetaData, который содержит метаданные базы данных, соединение с которой представляет данный объект SQLServerConnection.

getTransactionIsolation

Возвращает текущий уровень изоляции транзакции для данного объекта SQLServerConnection.

getTypeMap

Возвращает объект Map, связанный с данным объектом SQLServerConnection.

getWarnings

Возвращает первое предупреждение, указанное в отчетах вызовов этого объекта SQLServerConnection.

isClosed

Указывает, был ли закрыт данный объект SQLServerConnection.

isReadOnly

Указывает, находится ли данный объект SQLServerConnection в режиме только для чтения.

isValid

Указывает, верно ли, что данный объект SQLServerConnection не закрыт и до сих пор действителен.

nativeSQL

Преобразовывает заданную инструкцию SQL в соответствии с собственной грамматикой SQL на сервере базы данных.

prepareCall

Создает объект SQLServerCallableStatement для вызова хранимых процедур базы данных.

prepareStatement

Создает объект SQLServerPreparedStatement для отправки в базу данных параметризованных инструкций SQL.

releaseSavepoint

Удаляет заданный объект SQLServerSavepoint из текущей транзакции.

rollback

Отменяет все изменения, сделанные в текущей транзакции, и снимает в базе данных все блокировки, удерживаемые данным объектом SQLServerConnection.

setAutoCommit

Устанавливает режим автоматической фиксации для данного объекта SQLServerConnection в указанное состояние.

setCatalog

Задает указанное имя каталога для выбора подпространства базы данных данного объекта SQLServerConnection, в котором будет производиться работа.

setClientInfo

Задает значение для свойств сведений о клиенте.

setHoldability

Изменяет возможность сохранения объектов SQLServerResultSet, создаваемых с помощью данного объекта SQLServerSavepoint, устанавливая ее в заданное значение.

setReadOnly

Переводит данный объект SQLServerConnection в режим только для чтения, который является подсказкой драйверу JDBC включить оптимизацию базы данных.

setSavepoint

Создает в текущей транзакции неименованную точку сохранения и возвращает новый объект SQLServerSavepoint, который ее представляет.

setTransactionIsolation

Изменяет текущий уровень изоляции транзакций для данного объекта SQLServerConnection на заданный.

setTypeMap

Устанавливает указанный объект TypeMap в качестве сопоставления типов для данного объекта SQLServerConnection.
