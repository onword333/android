Метод setFetchDirection (ISQLServerStatement)

Определяет подсказку для драйвера JDBC Microsoft SQL Server как направление, в котором будут обрабатываться строки результирующего набора.

Примечание. Драйвер JDBC в настоящее время пропускает подсказку, определенную этим методом.

Синтаксис

public final void setFetchDirection(int nDir)

Параметры

nDir

Значение int, указывающее направление обработки строк, может принимать одно из следующих значений:

FETCH_FORWARD

FETCH_REVERSE

FETCH_UNKNOWN

Исключения

SQLServerException

Примечания

Этот метод setFetchDirection указывается методом setFetchDirection в интерфейсе java.sql.Statement.
