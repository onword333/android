Метод addConnectionEventListener (SQLServerPooledConnection)

Регистрирует заданный прослушиватель событий для получения уведомлений о событии в этом объекте SQLServerPooledConnection.

Синтаксис

public void addConnectionEventListener(javax.sql.ConnectionEventListener listener)

Параметры

listener

Объект ConnectionEventListener.

Примечания

Этот метод addConnectionEventListener указывается в методе addConnectionEventListener в интерфейсе javax.sql.PooledConnection.
