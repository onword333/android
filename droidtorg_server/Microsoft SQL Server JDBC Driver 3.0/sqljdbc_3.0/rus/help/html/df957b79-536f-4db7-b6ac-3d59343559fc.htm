Класс SQLServerXAResource

Представляет XAResource для управления распределенными транзакциями XA.

Пакет: com.microsoft.sqlserver.jdbc

Расширяет: java.lang.Object

Реализует: javax.transaction.xa.XAResource

Синтаксис

public class SQLServerXAResource

Примечания

Транзакции XA реализуются в SQL Server путем использования диспетчера распределенных транзакций (Майкрософт) (DTC). Класс SQLServerXAResource вызывает методы расширенной библиотеки SQL Server с именем named sqljdbc_xa.dll, которая служит интерфейсом для DTC. Вызовы XA, получаемые SQLServerXAResource (XA_START, XA_END, XA_PREPARE и т. д.), сопоставляются с вызовами соответствующих функций DTC.
