Класс SQLServerXAConnection

Представляет подключения JDBC, которые могут участвовать в распределенных транзакциях (XA).

Пакет: com.microsoft.sqlserver.jdbc

Расширяет: SQLServerPooledConnection

Реализует: javax.sql.XAConnection

Синтаксис

public class SQLServerXAConnection

Примечания

Объект SQLServerXAConnection можно прикрепить в распределенной транзакции посредством объекта SQLServerXAResource. Диспетчер транзакций, обычно входящий в сервер среднего уровня, управляет объектом SQLServerXAConnection посредством объекта SQLServerXAResource.

Примечание. Программисты приложений обычно не используют этот интерфейс непосредственно. Он используется главным образом диспетчером транзакций, работающим на сервере среднего уровня.
