Метод close (ISQLServerConnection)

Моментально освобождает для данного объекта SQLServerConnection базу данных и ресурсы JDBC, не дожидаясь их автоматического освобождения.

Синтаксис

public void close()

Исключения

SQLServerException

Примечания

Этот метод close указывается методом close в интерфейсе java.sql.Connection.

Вызов метода close в середине транзакции приводит к откату этой транзакции.
