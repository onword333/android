Метод setNull (java.lang.String, int, java.lang.String)

Устанавливает указанный параметр в значение NULL по заданному имени и типу параметра.

Синтаксис

public void setNull(java.lang.String sCol,
                    int nType,
                    java.lang.String sTypeName)

Параметры

sCol

Значение типа String, содержащее имя параметра.

nType

Код типа JDBC, определенный в java.sql.Types.

sTypeName

Значение типа String, которое указывает полное имя задаваемого параметра.

Исключения

SQLServerException

Примечания

Этот метод setNull указывается методом setNull в интерфейсе java.sql.CallableStatement.
