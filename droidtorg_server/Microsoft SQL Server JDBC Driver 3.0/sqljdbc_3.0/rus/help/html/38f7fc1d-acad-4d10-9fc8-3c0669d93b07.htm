Метод setMaxFieldSize (ISQLServerStatement)

Устанавливает ограничение числа байтов для максимального размера столбца SQLServerResultSet, в котором хранятся символьные или двоичные значения.

Синтаксис

public final void setMaxFieldSize(int max)

Параметры

max

Значение типа int, указывающее максимальное число байтов.

Исключения

SQLServerException

Примечания

Этот метод setMaxFieldSize указывается методом setMaxFieldSize в интерфейсе java.sql.Statement.
