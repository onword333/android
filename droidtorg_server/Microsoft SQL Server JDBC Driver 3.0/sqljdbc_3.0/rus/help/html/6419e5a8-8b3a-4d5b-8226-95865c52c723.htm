Метод getMoreResults (int)

Переходит к следующему результату этого объекта SQLServerStatement и обрабатывает все открытые объекты результирующего набора в соответствии с инструкциями, заданными указанным режимом.

Синтаксис

public final boolean getMoreResults(int mode)

Параметры

mode

Значение типа int, указывающее порядок обработки открытых объектов результирующего набора. Это должна быть одна из следующих констант:

CLOSE_CURRENT_RESULT

KEEP_CURRENT_RESULT (не поддерживается драйвером JDBC)

CLOSE_ALL_RESULTS

Возвращаемое значение

Значение true, если возвращенный результат является результирующим набором. В противном случае значение false.

Исключения

SQLServerException

Примечания

Этот метод getMoreResults указывается методом getMoreResults в интерфейсе java.sql.Statement.

Если метод getMoreResults вызван перед получением результатов, то он работает в соответствии со значением аргумента mode и переходит к следующему результату.

Примечание. Драйвер JDBC не поддерживает использование константы KEEP_CURRENT_RESULT. Если эта константа используется, вызывается исключение.
