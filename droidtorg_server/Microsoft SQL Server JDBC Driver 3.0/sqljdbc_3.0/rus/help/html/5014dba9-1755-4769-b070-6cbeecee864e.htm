Метод setDateTimeOffset (ISQLServerPreparedStatement)

Этот метод добавлен в версии 3.0 драйвера Microsoft SQL Server JDBC.

Устанавливает указанный столбец в значение Класс DateTimeOffset.

Синтаксис

public final void setDateTimeOffset(int n, microsoft.sql.DateTimeOffset x)

Параметры

n

Порядковый номер столбца (начиная с нуля).

x

Объект Класс DateTimeOffset.

Исключения

SQLServerException
