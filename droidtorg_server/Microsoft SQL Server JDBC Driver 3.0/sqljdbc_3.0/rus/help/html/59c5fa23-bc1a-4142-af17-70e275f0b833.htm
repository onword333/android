Метод setPortNumber (SQLServerDataSource)

Задает номер порта, используемого для обмена данными с SQL Server.

Синтаксис

public void setPortNumber(int portNumber)

Параметры

portNumber

Значение типа int, содержащее номер порта.

Примечания

Номер порта — это номер порта TCP/IP, который используется при открытии соединения с SQL Server через сокет. Если свойство portNumber не задано, то метод getPortNumber возвращает значение по умолчанию (1433).

Примечание. Метод setPortNumber не выполняет проверку допустимости диапазона для передаваемого значения порта. Можно передать недопустимый номер порта, например 9999, и это не вызовет ошибку.
