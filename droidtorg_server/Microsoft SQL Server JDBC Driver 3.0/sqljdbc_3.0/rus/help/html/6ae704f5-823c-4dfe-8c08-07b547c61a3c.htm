Метод getNCharacterStream (int)

Возвращает значение указанного параметра в виде объекта Reader по индексу параметра.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public final java.io.Reader getNCharacterStream(int parameterIndex)

Параметры

parameterIndex

Значение int, указывающее индекс параметра.

Возвращаемое значение

Объект Reader .

Исключения

SQLServerException

Примечания

Этот метод должен использоваться при доступе к параметрам NCHAR, NVARCHAR и LONGNVARCHAR.

Этот метод getNCharacterStream указывается с помощью метода getNCharacterStream в интерфейсе java.sql.CallableStatement.
