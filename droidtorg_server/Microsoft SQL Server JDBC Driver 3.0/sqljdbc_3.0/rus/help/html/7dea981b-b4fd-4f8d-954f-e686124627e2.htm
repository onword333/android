Метод getStatement (ISQLServerResultSet)

Извлекает объект SQLServerStatement, на основе которого был создан этот объект SQLServerResultSet.

Синтаксис

public java.sql.Statement getStatement()

Возвращаемое значение

Объект SQLServerStatement.

Исключения

SQLServerException

Примечания

Этот метод getStatement указывается с помощью метода getStatement в интерфейсе java.sql.ResultSet.

Если результирующий набор был получен другим образом, например методом SQLServerDatabaseMetaData, то этот метод возвращает значение NULL.
