Метод setDateTimeOffset(int, java.sql.DateTimeOffset) (SQLServerStatement)

Устанавливает указанный параметр в заданное значение типа DateTimeOffset.

Синтаксис

public void setDateTimeOffset(int parameterIndex, DateTimeOffset dateTime)

Параметры

parameterIndex

Индекс задаваемого столбца.

dateTimeOffset

Объект DateTimeOffset.

Исключения

SQLServerException

Примечания

DateTimeOffset имеет формат «ГГГГ-ММ-ДД ЧЧ-ММ-СС[.ннннннн] [+][-] ЧЧ:ММ». Следующая таблица приводится в справочных целях.

Тип SQL Вставка

datetime

Могут вставляться только значения в формате «ГГГГ-ММ-ДД чч:мм:сс[.ннн]»

smalldatetime

Могут вставляться только значения в формате «ГГГГ-ММ-ДД чч:мм:сс»

Time

Могут вставляться только значения в формате «чч:мм:сс[.ннннннн]»

Date

Могут вставляться только значения в формате «ГГГГ-ММ-ДД»

DateTime2

Могут вставляться только значения в формате «ГГГГ-ММ-ДД чч:мм:сс[.ннннннн]»
