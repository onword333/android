Метод execute (java.lang.String, java.lang.String[])

Выполняет заданную инструкцию SQL, которая может вернуть несколько результатов, и уведомляет драйвер Microsoft SQL Server JDBC о том, что автоматически сформированные ключи, отмеченные в заданном массиве, должны быть доступными для получения.

Синтаксис

public final boolean execute(java.lang.String sql,
                             java.lang.String[] columnNames)

Параметры

sql

Значение String, содержащее инструкцию SQL.

columnNames

Массив строк, которые указывают имена столбцов автоматически формируемых ключей, которые должны быть доступными.

Возвращаемое значение

Значение true, если первый результат является результирующим набором. В противном случае значение false.

Исключения

SQLServerException

Примечания

Этот метод execute указывается методом execute в интерфейсе java.sql.Statement.
