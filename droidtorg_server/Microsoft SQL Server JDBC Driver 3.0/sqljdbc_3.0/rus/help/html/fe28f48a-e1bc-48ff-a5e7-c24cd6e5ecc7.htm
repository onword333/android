Использование инструкций в SQL

При работе с данными в базе данных SQL Server с использованием инструкций драйвера JDBC Microsoft SQL Server и встроенных инструкций SQL можно использовать различные классы. Выбор используемого класса зависит от типа SQL-инструкции, которую необходимо выполнить.

Если инструкция SQL не содержит параметров IN, используется класс SQLServerStatement, в противном случае используется класс SQLServerPreparedStatement.

Примечание. Если необходимо использовать инструкцию SQL, которая содержит параметр IN и параметр OUT, необходимо реализовать ее в виде хранимой процедуры и вызвать ее с использованием класса SQLServerCallableStatement. Дополнительные сведения об использовании хранимых процедур см. в разделе Использование инструкций с хранимыми процедурами.
