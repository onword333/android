Метод setHostNameInCertificate (SQLServerDataSource)

Задает имя узла, используемого для проверки SSL-сертификата SQL Server.

Синтаксис

public void setHostNameInCertificate(java.lang.String hostNameInCertificate)

Параметры

hostNameInCertificate

Значение String, содержащее имя узла.

Примечания

Значение hostNameInCertificate используется для проверки SSL-сертификата SQL Server при использовании SSL-сертификата для шифрования уровня связи. По умолчанию задается значение NULL.

Если свойство hostNameInCertificate имеет значение NULL или оно не указано, драйвер JDBC Microsoft SQL Server использует значение свойства serverName для проверки с учетом SSL-сертификата SQL Server. Если для свойства hostNameInCertificate задано строковое значение или пустая строка "", драйвер использует это значение для проверки SSL-сертификата сервера.
