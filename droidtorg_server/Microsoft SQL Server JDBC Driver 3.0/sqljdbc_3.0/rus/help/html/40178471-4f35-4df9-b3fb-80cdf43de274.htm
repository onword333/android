Метод getFloat (int)

Получает значение заданного параметра в виде значения типа float на языке программирования Java по заданному индексу параметра.

Синтаксис

public float getFloat(int index)

Параметры

index

Значение int, указывающее индекс параметра.

Возвращаемое значение

Значение типа float.

Исключения

SQLServerException

Примечания

Этот метод getFloat указывается методом getFloat в интерфейсе java.sql.CallableStatement.

Этот метод возвращает все числовые типы с точностью float Java.
