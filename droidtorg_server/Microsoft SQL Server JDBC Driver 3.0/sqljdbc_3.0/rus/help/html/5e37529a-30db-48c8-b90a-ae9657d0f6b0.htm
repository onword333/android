Использование метаданных результирующего набора

В драйвере Microsoft SQL Server JDBC реализован класс SQLServerResultSetMetaData, позволяющий запрашивать в результирующем наборе сведения о содержащихся в нем столбцах. Этот класс содержит многочисленные методы, которые возвращают сведения в виде одного значения.

Для создания объекта SQLServerResultSetMetaData можно использовать метод getMetaData класса SQLServerResultSet.

В следующем примере в функцию передается открытое соединение с образцом базы данных SQL Server 2005 AdventureWorks, метод getMetaData method класса SQLServerResultSet возвращает объект SQLServerResultSetMetaData, а затем с помощью различных методов объекта SQLServerResultSetMetaData выводятся сведения об имени и типе данных столбцов, содержащихся в результирующем наборе.

public static void getResultSetMetaData(Connection con) {
   try {
      String SQL = "SELECT TOP 10 * FROM Person.Contact";
      Statement stmt = con.createStatement();
      ResultSet rs = stmt.executeQuery(SQL);
      ResultSetMetaData rsmd = rs.getMetaData();

      // Display the column name and type.
      int cols = rsmd.getColumnCount();
      for (int i = 1; i <= cols; i++) {
         System.out.println("NAME: " + rsmd.getColumnName(i) + " " + "TYPE: " + rsmd.getColumnTypeName(i));
      }
      rs.close();
      stmt.close();
   }
   catch (Exception e) {
      e.printStackTrace();
   }
}
