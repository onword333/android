Метод getServerName (SQLServerDataSource)

Возвращает имя экземпляра SQL Server.

Синтаксис

public java.lang.String getServerName()

Возвращаемое значение

Значение типа String, содержащее имя сервера, или значение NULL, если значение не задано.

Примечания

Именем сервера является сетевое имя узла целевого компьютера, где работает SQL Server. Если свойство getServerName не задано, метод getServerName возвращает значение NULL по умолчанию.
