Метод valueOf (java.sql.Timestamp, int)

Создает объект DateTimeOffset, представляющий точку во времени с определенным смещением от среднего времени по Гринвичу, заданного значением java.sql.Timestamp и значением, указывающим смещение в минутах.

Синтаксис

public static DateTimeOffset valueOf(java.sql.Timestamp timestamp, int minutesOffset)

Параметры

timestamp

Значение типа java.sql.Timestamp.

minutesOffset

Смещение в минутах.

Возвращаемое значение

Возвращает объект DateTimeOffset, представляющий точку во времени, заданную объектом java.sql.Timestamp и смещением в минутах от среднего времени по Гринвичу.
