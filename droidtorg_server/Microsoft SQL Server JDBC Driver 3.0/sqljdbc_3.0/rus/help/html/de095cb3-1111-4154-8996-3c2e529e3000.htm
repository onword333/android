Метод setSQLXML (SQLServerCallableStatement)

Задает указанный параметр для заданного объекта SQLXML.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public final void setSQLXML(java.lang.String parameterName,
                            java.sql.SQLXML xmlObject)

Параметры

parameterName

Значение String, определяющее имя параметра.

xmlObject

Объект SQLXML, содержащий значение параметра.

Исключения

SQLServerException

Примечания

Этот метод setSQLXML указывается с помощью метода setSQLXML в интерфейсе java.sql.CallableStatement.
