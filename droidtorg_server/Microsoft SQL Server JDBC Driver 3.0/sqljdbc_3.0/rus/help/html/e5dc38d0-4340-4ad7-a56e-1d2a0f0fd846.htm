Метод getPortNumber (SQLServerDataSource)

Возвращает текущий номер порта, используемый для обмена данными с SQL Server.

Синтаксис

public int getPortNumber()

Возвращаемое значение

Значение типа int, содержащее текущий номер порта.

Примечания

Номер порта — это номер порта TCP/IP, который используется при открытии соединения с SQL Server через сокет. Если свойство portNumber не задано, то метод getPortNumber возвращает значение по умолчанию (1433).

Примечание. Метод setPortNumber не выполняет проверку диапазона для переданного значения порта. Можно передавать недопустимые номера порта, например 99999, и это не вызовет ошибку.
