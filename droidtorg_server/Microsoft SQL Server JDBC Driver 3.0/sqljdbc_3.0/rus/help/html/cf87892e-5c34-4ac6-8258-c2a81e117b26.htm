Интерфейс ISQLServerPreparedStatement

Представляет базовую реализацию функциональности подготовленной инструкции JDBC. Этот интерфейс добавлен в версии 3.0 драйвера SQL Server JDBC.

Пакет: com.microsoft.sqlserver.jdbc

Расширяет: java.sql.PreparedStatement, ISQLServerStatement

Синтаксис

public interface ISQLServerPreparedStatement

Примечания

Этот интерфейс реализуется классом Класс SQLServerPreparedStatement.

Этот интерфейс предоставляет следующие методы SQL Server, относящиеся к драйверу JDBC.

Метод Дополнительные сведения см. в разделе

public void setDateTimeOffset(int, microsoft.sql.DateTimeOffset)
