Метод truncate (SQLServerBlob)

Выполняет усечение большого двоичного объекта до заданной длины.

Синтаксис

public void truncate(long len)

Параметры

len

Новая длина большого двоичного объекта.

Исключения

java.sql.SQLException

Примечания

Этот метод truncate указывается с помощью метода truncate в интерфейсе java.sql.Blob.
