Общие сведения о драйвере JDBC

Драйвер Microsoft SQL Server JDBC совместим со спецификацией Java Database Connectivity 4.0, который предоставляет стабильный доступ к данным в базах данных Microsoft SQL Server. Драйвер JDBC имеет доступ ко многим функциям SQL Server 2005, включая зеркальное отображение базы данных, типы данных xml, определяемые пользователем типы данных и типы данных большого объема. Драйвер поддерживает изоляцию моментальных снимков для транзакций. Помимо этого, драйвер JDBC поддерживает встроенную проверку подлинности. В драйвере JDBC для SQL Server версии 3.0 добавлена поддержка некоторых функций SQL Server 2008. Дополнительные сведения см. в разделе Новые возможности драйвера JDBC.

В подразделах данного раздела приводятся общие сведения о драйвере JDBC, включая системные требования, необходимые для его использования, а также описание его использования и ссылки для получения дополнительных сведений.
