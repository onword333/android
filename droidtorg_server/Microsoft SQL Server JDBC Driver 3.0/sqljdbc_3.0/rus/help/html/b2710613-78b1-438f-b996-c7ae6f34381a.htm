Метод getXAConnection ()

Устанавливает физическое соединение с базой данных, которое будет использоваться в распределенной транзакции.

Синтаксис

public javax.sql.XAConnection getXAConnection()

Возвращаемое значение

Объект XAConnection.

Исключения

java.sql.SQLException

Примечания

Этот метод getXAConnection указывается в методе getXAConnection в интерфейсе javax.sql.XADataSource.

Примечание. Этот метод обычно вызывается в реализациях пула соединений XA, а не в обычном коде приложений JDBC.
