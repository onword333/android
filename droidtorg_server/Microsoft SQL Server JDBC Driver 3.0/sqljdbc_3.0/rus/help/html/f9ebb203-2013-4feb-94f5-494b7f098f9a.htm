Методы

Имя Описание

getParameterClassName

Возвращает полное имя класса Java, экземпляры которого должны передаваться методу setObject класса SQLServerPreparedStatement.

getParameterCount

Возвращает число параметров в объекте SQLServerPreparedStatement, для которого данный объект SQLServerParameterMetaData содержит данные.

getParameterMode

Возвращает режим указанного параметра.

getParameterType

Возвращает тип SQL указанного параметра.

getParameterTypeName

Возвращает имя определяемого базой данных типа для указанного параметра.

getPrecision

Возвращает число десятичных разрядов для указанного параметра.

getScale

Возвращает число цифр справа от десятичной запятой для указанного параметра.

isNullable

Возвращает значение, определяющее, допустимы ли значения NULL для указанного параметра.

isSigned

Возвращает значение, определяющее, могут ли быть значения указанного параметра числами со знаком.
