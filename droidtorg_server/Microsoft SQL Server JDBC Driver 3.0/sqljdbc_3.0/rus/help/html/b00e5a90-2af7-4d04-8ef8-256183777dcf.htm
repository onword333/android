Класс SQLServerConnectionPoolDataSource

Представляет физические соединения с базой данных для диспетчеров пулов соединений.

Пакет: com.microsoft.sqlserver.jdbc

Расширяет: SQLServerDataSource

Реализует: javax.sql.ConnectionPoolDataSource

Синтаксис

public class SQLServerConnectionPoolDataSource

Примечания

Класс SQLServerConnectionPoolDataSource обычно используется в средах Java Application Server, поддерживающих встроенные пулы соединений и требующих для установки физических соединений объект ConnectionPoolDataSource, например на серверах приложений платформы Java Enterprise Edition (Java EE), которые обеспечивают организацию пулов соединений по спецификациям JDBC 3.0 API.
