Метод updateDateTimeOffset (ISQLServerResultSet)

Этот метод добавлен в версии 3.0 драйвера Microsoft SQL Server JDBC.

Обновляет значение Класс DateTimeOffset в указанном столбце.

Список перегрузок

Имя Описание

updateDateTimeOffset(int, microsoft.sql.DateTimeOffset)

Обновляет указанный столбец значением Класс DateTimeOffset по заданному порядковому номеру столбца (начиная с нуля).

updateDateTimeOffset(string, microsoft.sql.DateTimeOffset)

Обновляет указанный столбец значением Класс DateTimeOffset по заданному имени столбца.
