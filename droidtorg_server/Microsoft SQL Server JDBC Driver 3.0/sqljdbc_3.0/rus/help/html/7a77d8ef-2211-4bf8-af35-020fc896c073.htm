Метод setLogWriter (SQLServerDataSource)

Этот метод предназначен только для внутреннего использования. Дополнительные сведения о ведении журнала см. в разделе Трассировка операций драйвера.

Синтаксис

public void setLogWriter(java.io.PrintWriter out)

Параметры

out

Объект PrintWriter.

Примечания

Этот метод setLogWriter указывается методом setLogWriter в интерфейсе javax.sql.DataSource.
