Конструктор SQLServerSavepoint (SQLServerConnection, java.lang.StringName)

Инициализирует новый экземпляр класса SQLServerException на основе заданного соединения и имени.

Синтаксис

public SQLServerSavepoint(SQLServerConnection con,
                          java.lang.String sName)

Параметры

con

Объект SQLServerConnection.

sName

Значение типа String, содержащее имя точки сохранения.
