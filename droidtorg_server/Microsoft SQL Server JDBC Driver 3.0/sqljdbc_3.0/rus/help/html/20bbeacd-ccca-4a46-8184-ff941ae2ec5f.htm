Класс DateTimeOffset

Этот тип представляет объект типа microsoft.sql.DateTimeOffset и является типом Java, соответствующим типу данных SQL Server datetimeoffset.

Расширяет: java.lang.Object

Реализует: java.lang.Comparable<DateTimeOffset>, java.io.Serializable
