Метод getNString (java.lang.String) (ISQLServerResultSet)

Извлекает значение указанного столбца в текущей строке объекта SQLServerResultSet в виде объекта java.lang.String.

Примечание. Эта функция введена в драйвере JDBC Microsoft SQL Server версии 2.0.

Синтаксис

public java.lang.String getNString(java.lang.String columnLabel)

Параметры

columnLabel

Значение String, содержащее метку столбца.

Возвращаемое значение

Объект String.

Исключения

SQLServerException

Примечания

Этот метод getNString указывается с помощью метода getNString в интерфейсе java.sql.SQLServerResultSet.

Этот метод может использоваться для получения значения из столбца nvarchar, nchar, nvarchar(max), ntext или xml в текущей строке объекта SQLServerResultSet. При попытке использования этого метода для получения значений других типов данных возникнет исключение.
