droidtorg (приложение для android)
Автоматизация процесса сбора заявок (пресселинг).
Позволяет торговым представителям сформировать заявку, иными словами - список необходимого товара, и отправить ее продавцу одним нажатием на кнопку.
Имеет справочники номенклатуры, цен, ед. изм., журнал заявок. 
Есть возможность:
+ обновить с сервера остатки номенклатуры
+ обновить с сервера список клиентов
+ обновить с сервера долги клиентов
+ получить остаток в режиме онлайн по конкретной номенклатуре
+ сформировать отчет по собранным заявкам
+ отправить через интернет одну и сразу пачку заявок.

droidtorg_server (сервер написан на java)
Сервер для droidtorg. 
Принимает заявки отправленные пользователями и вносит их в базу данных 1С.
